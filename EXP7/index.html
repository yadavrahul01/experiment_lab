Understanding Event Handling

1. Inline HTML Attributes

This is the oldest method where we add the JavaScript directly inside the HTML tag (e.g., onclick="doSomething()"). It is generally considered bad practice today because it mixes HTML structure with JavaScript logic.

2. DOM Element Properties

Here, we select the element in JavaScript and assign a function to its event property (like element.onclick = function). It's cleaner than inline handlers, but you can only assign one function per event type on a single element.

3. Event Listeners (Best Practice)

The addEventListener() method is the modern standard for handling events. It allows you to attach multiple listeners to a single element without overwriting existing ones, giving you the most control over how events propagate.
